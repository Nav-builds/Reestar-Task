# Reestar-Task
🍽️ Front Menu Page (HTML & CSS)

This project is a responsive front menu page built using HTML5 and CSS3. It showcases a modern card-based layout designed with Flexbox for alignment and responsiveness, along with smooth hover effects for better user interaction.

🚀 Features

Structured semantic HTML5 layout

Responsive design using CSS Flexbox

Interactive menu cards with hover effects

Clean and modern UI styling

Lightweight and beginner-friendly project

🛠️ Technologies Used

HTML5

CSS3 (Flexbox, transitions, hover effects)

🎯 Purpose

This project was created as a practice exercise to improve layout structuring, Flexbox positioning, and UI styling skills using pure HTML and CSS without any frameworks.

📱 Responsiveness

The layout adjusts to different screen sizes using flexible containers and alignment techniques provided by Flexbox
